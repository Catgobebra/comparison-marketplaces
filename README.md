# Система принятия решений на основе взвешенных критериев

**Браузерное расширение для Chrome** (Manifest V3)  
Сравнение товаров с Ozon, Wildberries и Яндекс.Маркет в одной таблице с возможностью ручной настройки весов критериев, drag-and-drop сортировки характеристик, визуализации (линейные и радиальные диаграммы) и рекомендаций.

## Возможности
- Добавление товаров по URL или кнопкой на странице
- Автоматический парсинг данных (Ozon, Wildberries, Яндекс.Маркет)
- Нормализация и объединение характеристик
- Ручная настройка весов критериев
- Выделение лучших значений
- Drag-and-drop сортировка характеристик
- Визуализация: линейные и радиальные диаграммы
- Категории товаров (пользовательские)
- Светлая / тёмная тема
- Уведомления (Snackbar)
